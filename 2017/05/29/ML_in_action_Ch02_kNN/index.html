Ch02. kNN

kNN算法概述

k近邻法（k-nearest neighbor, k-NN）是一种基本的分类方法。首先存在一个样本数据集合，即训练样本集，数据集之中每个数据有其对应的标签即所属类别。当输入一个新的数据后，将新数据的各个特征属性值与训练样本的各个数据的各个属性值进行比较，提取样本最为相似的数据，选择前k个最相似的数据，并将k个相似的数据中，同一个类别数据出现最多的类别作为输入样本的类别。

这里涉及了两个基本问题，一是什么叫和样本的数据相似，二是这个k值怎么选择。

对于第一个问题，通用的解决方法是，把特征属性的值转化为数字，把一个数据样本作为空间中的一个点，计算输入样本与训练样本的空间欧氏距离（Euclidean distance）即最短的线段距离，

$\sqrt{(\displaystyle\sum_{l=1}^{n} |x_i^l-x_j^l|^2)}$

距离的大小即可用来比较相似程度。衡量距离也可使用更为一般的Lp距离和Minkowski距离，具体的可以参照漫谈：机器学习中距离和相似性度量方法 。

对于第二个问题，没有绝对有效的确定方法，k值较小则相当于较小的邻域进行训练，容易发生过拟合；选择较大的k值，学习的近似误差会增大，与输入样本相差较远和分类样本不相似的点也会对分类造成影响，一般k值选择较小，采用交叉验证法来选取最优的k值。

归一化数值处理

在计算某两个样本间距离时 $\sqrt{(0-67)^2+(20000-32000)^2+(1.1-0.1)^2}$ ，会发现这个值的大小受到差值最大的特征值的影响极大，也就是说在本例中每年的飞行里程数这个属性值的影响程度要远远大于另外两个属性，而其原因仅仅是因为其数值远大于其他值。不过目标用户认为该属性值应和其余属性值同等重要，不应该让里程数如此严重的影响到计算结果。

在处理这种情况时，通常采取的措施是把数值归一化 ，将数值处理到变为01或-11之间的一个数，公式为：

$newValue = (oldValue-min)/(max - min)$

那么在实现时先得到每个属性值的max和min，再减一下除一下就好
