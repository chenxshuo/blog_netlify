逐块读取文本文件

当数据文件的行数十分多时，可以分块读取或者分块对文件进行迭代，常用的参数为nrows设置读取的行数，和chunksize, get_chunk

使用chunksize和get_chunk参数，分块读取和对每一块数据进行迭代计算

将数据写出到文本格式

使用DataFrame的to_csv函数，可以将数据写入到以逗号分隔的文件中去，同样的可以用sep参数指定分隔符，可以用na_rep指定缺失值的值，可以写出行和列的标签也可以不写出，可以指定顺序排列

使用Series的to_csv方法也可以存储

JSON数据

JSON即JavaScript Object Notation，python自带有json的包，常用json.load, json.loads, json.dumps

Note that dump and load convert between files and objects, while dumps and loads convert between strings and objects.
